# links

Official Mossland link hub for `https://links.moss.land`.

This repository contains the static landing page for Mossland official links.
It is deployed with AWS Amplify and served at `https://links.moss.land`.

## Files

- `index.html` - main page
- `amplify.yml` - Amplify build and deploy config
- `customHttp.yml` - Amplify custom headers
